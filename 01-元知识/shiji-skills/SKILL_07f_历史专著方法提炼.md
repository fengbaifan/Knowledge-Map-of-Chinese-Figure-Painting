# SKILL_07f: 从历史专著中提炼研究方法论

> 系统化地从历史学术专著中提炼可复用的研究方法，建立方法论体系

---

## 一、SKILL 概述

### 1.1 核心目标

从历史学术专著中：
- **识别**：发现作者使用的显性和隐性研究方法
- **提炼**：将具体操作抽象为可复用的方法论
- **分类**：建立方法的层次和类型体系
- **验证**：通过案例检验方法的有效性
- **应用**：将方法应用于新的研究对象

### 1.2 适用范围

**最适合**：
- 学术性强、方法论自觉的历史专著
- 有明确史料批判和推理过程的著作
- 作者对研究方法有元层次反思的作品

**次适合**：
- 普及性历史著作（方法隐性，需深度挖掘）
- 传统考据类著作（方法单一但严谨）

**不适合**：
- 纯粹史料汇编（无方法论）
- 历史小说（虚构成分多）

### 1.3 方法来源

本SKILL基于以下实践经验总结：
- **李开元四部曲**（《秦谜》《秦崩》《楚亡》《汉兴》）：提炼出30+种历史推理方法
- **张诗坪、胡可奇《安史之乱：历史、宣传与神话》**：提炼出8种战争史与财政史研究方法

两次实践验证了本SKILL的有效性和可复用性。

## 二、方法提炼的六大步骤

### 步骤1：全书通读与结构分析

**目标**：理解作者的研究框架和整体方法论

**操作**：
1. **快速通读**：了解全书主题、时间跨度、核心论点
2. **结构分析**：
   - 章节组织逻辑（时间/主题/人物）
   - 是否有专门的方法论章节
   - 前言/后记中的方法论说明
3. **标注关键段落**：
   - 作者明确说明方法的地方
   - 典型案例分析段落
   - 对传统研究的批判或创新之处

**输出**：
- 全书结构图谱
- 核心论点列表
- 方法论线索清单

**示例1（李开元四部曲）**：
```
《秦谜》（2009）：
- 结构：7个历史谜题 + 方法论总结
- 核心论点：秦始皇非吕不韦之子
- 方法论线索：前言明确提出"历史推理七法"

《汉兴》（2021）：
- 结构：吕后专政时期的政治史
- 核心论点："一切历史都是推想"
- 方法论线索：专门章节讨论"史真-史料-史实-史著"框架
```

**示例2（《安史之乱》）**：
```
《安史之乱：历史、宣传与神话》（2023）：
- 结构：30章按时间顺序叙述 + 序言阐述写作目标
- 核心论点：安史之乱的失败根源是财政崩溃，非单纯军事或道德问题
- 方法论线索：
  * 副标题"历史、宣传与神话"暗示解构立场
  * 序言："复盘棋手的考量，而非越俎代庖"（决策者视角）
  * 序言设问："昏君佞臣真的宁可战败也要拖后腿？"（质疑传统解释）
  * 缘起："对战役的描绘...接近西方现代学术界"（借鉴西方方法）
  * 缘起："财政数据的核算"、"后勤模型"（量化方法）
- 关键发现：作者自述写作目标和方法论立场，是识别方法的重要线索
```

### 步骤2：方法识别与分类

**目标**：识别作者使用的所有研究方法（显性+隐性）

**操作**：
1. **显性方法识别**：
   - 作者明确命名的方法
   - 在方法论章节中阐述的方法
   - 反复使用的固定套路
2. **隐性方法挖掘**：
   - 作者未命名但反复使用的操作
   - 案例分析中体现的思维方式
   - 与传统方法的差异之处
3. **方法分类**：
   - 按研究对象：史料批判/真实性推理/传播链分析
   - 按思维类型：演绎/归纳/类比/排除
   - 按学科借鉴：法律/医学/军事/心理学

**输出**：
- 方法清单（显性+隐性）
- 方法分类体系
- 每种方法的初步定义

**示例1（李开元四部曲）**：
```
显性方法（作者明确命名）：
- 《秦谜》：排除法、动机分析法、法律鉴定法...（7种）
- 《汉兴》："一切历史都是推想"（元理论）

隐性方法（从案例中提炼）：
- 《秦崩》：多源拼图法（作者未命名，但在沙丘政变分析中反复使用）
- 《楚亡》：战争逻辑分析法（在破釜沉舟等案例中隐含使用）
```

**示例2（《安史之乱》）**：
```
显性方法（作者明确说明）：
- 宣传话语解构："摈弃廉价的宣传"（缘起）
- 决策者视角复盘："复盘棋手的考量"（序言）
- 借鉴西方方法："接近西方现代学术界对古代战争的描绘方式"（序言）
- 学术成果整合：大量引用李碧妍、仇鹿鸣等专家研究

隐性方法（从分析中提炼）：
- 后勤与财政计算法：第5章标题"'不惜'与'代价'的矛盾"暗示，正文有大量财政计算
- 战场现场重建法：各战役章节（第9章灵宝、第17章香积寺等）有详细地形和战术分析
- 社会基础分析法：第1章分析河北对安禄山的支持，引用仇鹿鸣"社会基础"观点
- 多维因果分析法：拒绝单一解释（如"昏君导致战败"），从政治、军事、经济多维分析
- 制度与人性互动法：第19章"比平叛更重要的是什么？"分析功高震主困境

方法分类：
- 史料批判类：宣传话语解构法
- 量化分析类：后勤与财政计算法
- 战争史类：战场现场重建法（借鉴西方古典学）
- 决策分析类：决策者视角复盘法、制度与人性互动法
- 系统分析类：社会基础分析法、多维因果分析法
- 学术整合类：学术成果整合创新法
```

**关键发现**：
1. **副标题是重要线索**："历史、宣传与神话"直接揭示了"解构宣传"方法
2. **章节标题隐含方法**：如"'不惜'与'代价'的矛盾"暗示财政计算，"比平叛更重要的是什么？"暗示决策困境分析
3. **序言是方法论宣言**：作者自述写作目标和方法立场，是识别方法的最重要文本
4. **评论人的评价**：罗振宇、陆大鹏等人的评语（"接近西方学术"、"像戈兹沃西"）直接点明方法特征

### 步骤3：方法详细定义

**目标**：将识别出的方法抽象为可复用的方法论

**操作**：
对每种方法进行系统化定义，包括：

1. **方法名称**：简洁准确（3-8个字）
2. **核心定义**：一句话说明方法的本质（20-50字）
3. **理论基础**：方法的学理依据（3-5条）
4. **应用步骤**：可操作的流程（5-7步）
5. **典型案例**：从原著中提取1-3个案例
6. **适用范围**：最适用/次适用/不适用的情况
7. **局限性**：方法的前提假设和局限

**输出**：
- 每种方法的标准化定义文档
- 方法之间的关系图谱（继承/深化/组合）

**模板**：
```markdown
### 方法X：[方法名称]

**定义**：[一句话核心定义]

**理论基础**：
- [理论依据1]
- [理论依据2]
- [理论依据3]

**应用步骤**：
1. [步骤1]
2. [步骤2]
...

**典型案例**：[从原著提取]

**适用范围**：
- 最适用：[情况]
- 次适用：[情况]
- 不适用：[情况]

**局限性**：[说明]
```

**示例1（李开元方法）**：
```markdown
### 方法1：排除法

**定义**：通过排除不可能的选项，缩小可能性范围，逼近历史真相

**理论基础**：
- 历史事件的可能性是有限的
- 逻辑矛盾的选项可以排除
- 剩余可能性中，最简单合理的最可能为真

**应用步骤**：
1. 列举所有可能的历史解释
2. 用史料证据排除矛盾选项
3. 用逻辑推理排除不合理选项
4. 用常识排除违背人性的选项
5. 评估剩余选项的可能性

**典型案例**：秦始皇身世之谜
- 可能性：吕不韦之子 vs 子楚之子
- 排除依据：孕期医学常识、法律继承规则、动机分析
- 结论：子楚之子可能性更大

**适用范围**：
- 最适用：有多种历史解释的谜题
- 次适用：史料矛盾的事件
- 不适用：完全无史料的事件

**局限性**：无法提供积极证明，只能缩小范围
```

**示例2（《安史之乱》方法）**：
```markdown
### 方法：后勤与财政计算法

**定义**：通过精确计算军队的粮饷、后勤、财政成本，验证战争记载的合理性，揭示战略选择的经济约束

**理论基础**：
- 战争不仅是战术和战略的博弈，更是后勤和财政的较量
- 经济和财政状况极大地限制了决策者的选择空间
- 唐代财政和军事制度文献丰富，使得量化计算成为可能
- 欧美古典战争史研究重视后勤和经济因素（借鉴经验）

**应用步骤**：
1. 收集基础数据：军队规模、粮饷标准、运输成本、财政收入
2. 建立计算模型：
   - 军队总粮饷需求 = 人数 × 人均消耗 + 马匹 × 马匹消耗
   - 运输成本 = 粮食数量 × 距离 × 损耗率
   - 财政压力 = 军费开支 / 财政总收入
3. 计算关键节点：特定战役的后勤可行性、长期战争的财政可持续性
4. 验证史料记载：战略选择在财政上是否可行？
5. 揭示隐藏约束：决策者面临的经济压力和"不得不"的选择

**典型案例**：唐军"催令决战"的财政逻辑
- 传统解释：昏君佞臣私心作梗
- 财政计算：
  * 10万唐军每月消耗约6300吨粮食
  * 运输300里损耗率30-50%，实际需从后方发出9000-12000吨
  * 唐廷失去河北、河南税源，财政无法支撑长期战争
- 新解释："催令决战"非单纯政治私心，而是财政压力下的无奈选择
- 结论：揭示了"'不惜'与'代价'的矛盾"

**适用范围**：
- 最适用：有详细财政和军事制度文献的时期（如唐代）
- 次适用：有部分数据可供估算的时期
- 不适用：完全缺乏财政数据的时期

**局限性**：
- 依赖准确的基础数据（古代数据可能有缺失或误差）
- 计算模型的假设可能不完全符合历史实际
- 过度依赖经济决定论可能忽视其他因素（如士气、偶然性）

**方法来源**：
- 原著引用：李锦绣《唐代财政史稿》、《太白阴经》等
- 作者创新：建立可操作的后勤计算模型
- 提炼者命名："后勤与财政计算法"（原著未明确命名）
```

**关键改进点**：
1. **增加"方法来源"字段**：说明方法是作者原创、借鉴还是提炼者命名
2. **量化方法需详细公式**：如财政计算的具体公式
3. **强调与传统解释的对比**：突出方法的创新价值
4. **案例要有"前-后"对比**：传统解释 → 应用新方法 → 新解释

### 步骤3.5：关键文本深度分析（新增步骤）

**目标**：深度分析书中最能体现方法论的关键文本片段

**为何新增此步骤**：
- 《安史之乱》的实践表明，某些关键文本（序言、章节标题、设问句）浓缩了核心方法
- 这些文本往往比正文更直接地揭示方法论立场
- 深度分析关键文本可以快速定位核心方法

**操作**：
1. **识别关键文本类型**：
   - **序言/前言/后记**：作者自述写作动机和方法论立场
   - **副标题**：往往浓缩核心方法（如"历史、宣传与神话"）
   - **章节标题**：隐含方法论视角（如"'不惜'与'代价'的矛盾"）
   - **设问句**：揭示作者质疑的传统解释（如"昏君真的宁可战败？"）
   - **他人评论**：推荐序、书评中对方法的评价
2. **深度分析关键文本**：
   - 提取关键词（如"复盘"、"摈弃宣传"、"量化"、"西方方法"）
   - 识别立场性表述（如"不越俎代庖"、"不低估古人智慧"）
   - 发现隐含的方法论对立（如"vs《资治通鉴》框架"）
3. **建立关键词-方法对应表**：
   - "复盘棋手的考量" → 决策者视角复盘法
   - "摈弃宣传" → 宣传话语解构法
   - "财政数据核算" → 后勤与财政计算法
   - "接近西方学术" → 战场现场重建法（借鉴西方古典战争史）
4. **验证对应关系**：
   - 在正文中寻找对应方法的具体应用
   - 确认关键词-方法对应关系的准确性

**输出**：
- 关键文本摘录集（带标注）
- 关键词-方法对应表
- 方法论立场总结

**示例（《安史之乱》）**：

**关键文本1：序言中的方法论宣言**
```
原文："用下棋作比喻，就是尽量复盘一千两百多年前以天下为棋盘的一盘大棋中，
诸多顶尖棋手所下的每一步棋背后的考量，而不是'精神穿越'回去越俎代庖的给古人摆棋子。"

关键词提取：
- "复盘" → 决策者视角复盘法
- "棋手的考量" → 理解决策逻辑，非事后批判
- "不越俎代庖" → 拒绝用上帝视角批判古人

方法论立场：站在历史决策者的立场，理解其约束和困境
```

**关键文本2：序言设问句**
```
原文："唐军多次在战略形势大好时突遭翻盘，真是昏君佞臣因为私心一再作梗吗？"
"通过计算两军的粮饷和后勤状况，我们能得出什么新结论？"

关键词提取：
- "真是...吗？" → 质疑传统解释（忠奸二元论）
- "计算粮饷和后勤" → 后勤与财政计算法
- "新结论" → 方法创新的承诺

方法论立场：用量化分析替代道德批判
```

**关键文本3：副标题的深意**
```
副标题："历史、宣传与神话"

三个层次：
- "历史" = 真相（研究目标）
- "宣传" = 官方话语的政治建构（需要解构的对象）
- "神话" = 过度宣传的产物（如"7000 vs 20万"）

隐含方法：宣传话语解构法
```

**关键文本4：引用李碧妍的话**
```
原文："千万不要低估古人的政治智慧，更不要随意褒贬，
我们能做的也许只是尽自己的努力站在他们的立场，试图了解他们的所行所为，仅此而已。"

方法论立场：
- 理解古人的困境和智慧
- 拒绝道德化批判
- 这是决策者视角复盘法的理论基础
```

**关键文本5：章节标题分析**
```
第5章："'不惜'与'代价'的矛盾：大唐名将们的无解难题"
- "'不惜'与'代价'"：朝廷希望"不惜代价"取胜，但财政上"付不起代价"
- 隐含方法：后勤与财政计算法
- "无解难题"：结构性困境分析

第19章："没有乘胜追击的唐军：比平叛更重要的是什么？"
- 设问式标题：挑战读者预设（应该追击）
- 隐含方法：决策者视角复盘法、制度与人性互动分析法
- 答案：政治稳定、功高震主的自保
```

**方法应用价值**：
- 快速定位核心方法（节省80%阅读时间）
- 理解作者的方法论自觉程度
- 发现隐藏的方法论立场

### 步骤4：方法演进分析

**目标**：追踪作者方法论的发展轨迹（如果有系列著作或可与其他学者对比）

**操作**：

**情况A：同一作者的系列著作**
1. **继承关系**：后作如何继承前作的方法
2. **深化方向**：后作如何深化前作的方法
3. **新增领域**：后作开拓的新方法领域
4. **理论突破**：方法论的元层次提升

**情况B：不同作者/学派的方法对比**（新增）
1. **研究对象差异**：不同历史时期/主题是否导致方法差异？
2. **学科背景差异**：作者的学科训练如何影响方法选择？
3. **方法互补性**：不同作者的方法能否结合使用？
4. **方法创新点对比**：各自突破了哪些传统史学的局限？

**输出**：
- 方法演进图谱
- 不同作品的方法论对比表
- 作者方法论思想的发展脉络

**示例A：李开元四部曲演进**（同一作者）：
```
第一阶段《秦谜》（2009）：
- 建立7种基础方法
- 特点：谨慎、经验性

第二阶段《秦崩》（2015）：
- 深化：从单源到多源（多源拼图法）
- 新增：时空框架（时间线+行踪重建）

第三阶段《楚亡》（2015）：
- 专业化：战争逻辑、性格一致性
- 具体化：现场复活法

## 三、方法提炼的质量标准

### 3.1 方法定义的SMART原则

- **S (Specific)**: 方法定义具体明确，非模糊概念
- **M (Measurable)**: 方法应用效果可评估
- **A (Actionable)**: 方法有可操作的步骤
- **R (Reusable)**: 方法可应用于新对象
- **T (Theoretically-grounded)**: 方法有学理依据

### 3.2 文档质量标准

- **完整性**：方法定义的7要素齐全（名称/定义/理论/步骤/案例/范围/局限）
- **准确性**：忠实反映原著的方法，非过度解读
- **系统性**：方法之间的关系清晰（分类/层次/演进）
- **可读性**：结构清晰，术语准确，示例丰富
- **可用性**：他人可依据文档应用方法

### 3.3 常见陷阱

❌ **过度抽象**：将方法定义得太宽泛，失去可操作性
❌ **过度具体**：将方法绑定在特定案例上，无法推广
❌ **混淆层次**：将具体操作、方法、理论混为一谈
❌ **忽视局限**：只说方法的优点，不说适用范围和局限
❌ **脱离原著**：过度发挥，添加原著没有的内容

## 五、快速上手指南

### 5.1 30分钟快速提炼

**目标**：快速掌握一本书的核心方法

**流程**：
1. **阅读前言和目录**（5分钟）
   - 了解作者的研究问题和方法论自述
2. **阅读结论章节**（5分钟）
   - 看作者如何总结方法和贡献
3. **快速浏览2-3个核心案例**（15分钟）
   - 标注作者的推理过程
   - 识别反复出现的操作
4. **提炼3-5个核心方法**（5分钟）
   - 用一句话定义每个方法
   - 记录典型应用场景

**输出**：核心方法清单（初稿）

### 5.2 3小时深度提炼

**目标**：系统提炼一本书的完整方法论

**流程**：
1. **全书通读**（1.5小时）
   - 标注所有方法论相关段落
   - 记录典型案例
2. **方法识别与定义**（1小时）
   - 列举所有方法（显性+隐性）
   - 为每个方法撰写初步定义
3. **文档化**（0.5小时）
   - 整理为结构化文档
   - 补充案例和说明

**输出**：完整的方法论报告（初稿）

### 5.3 多书系列提炼

**目标**：提炼作者的完整方法论体系（如四部曲）

**流程**：
1. **分书提炼**（每本3小时）
   - 按单书深度提炼流程
2. **方法演进分析**（2小时）
   - 对比不同书的方法
   - 识别继承、深化、创新
3. **整合与系统化**（3小时）
   - 建立统一的方法分类
   - 绘制演进图谱
   - 撰写总体报告
4. **验证与完善**（2小时）
   - 案例验证
   - 文档完善

**总时长**：约20小时（4本书）
**输出**：完整的方法论体系文档

## 六、常见问题 (FAQ)

### Q1: 如何判断一本书是否值得提炼方法？

**判断标准**：
1. 作者是否有明确的方法论自觉？
2. 书中是否有典型案例展示方法应用？
3. 方法是否具有一定的普遍性（非仅适用于该书主题）？
4. 学术界对该书的评价如何？

**建议**：
- 优先选择学术性强、影响力大的专著
- 选择有方法论创新的作品
- 选择与自己研究方向相关的书

### Q2: 如何区分"方法"和"具体操作"？

**方法**：
- 可抽象为一般性原则
- 可应用于不同的研究对象
- 有明确的理论依据

**具体操作**：
- 绑定在特定案例上
- 难以推广到其他对象
- 是方法的具体实例

**示例**：
- 方法：排除法（可应用于任何历史谜题）
- 具体操作：用医学常识排除秦始皇是吕不韦之子（仅适用于该案例）

### Q3: 提炼的方法太多怎么办？

**建议**：
1. **分层次**：区分基础方法/深化方法/扩展方法
2. **分类型**：按研究对象/思维类型/学科来源分类
3. **抓重点**：识别核心方法（高频+高影响）
4. **做索引**：建立方法索引和快速查找表

**李开元案例**：
- 30+种方法，分为基础7种+深化12种+扩展10+种
- 核心方法：排除法、动机分析法、多源拼图法等（高频应用）

### Q4: 如何处理作者未明确命名的隐性方法？

**操作**：
1. **识别模式**：从案例中识别反复出现的操作模式
2. **抽象定义**：将模式抽象为方法定义
3. **命名**：为方法取一个准确简洁的名称
4. **标注来源**：注明这是提炼者命名，非作者原命名

**示例**：
- 李开元在《秦崩》中反复用不同来源的史料拼接历史
- 提炼者命名为"多源拼图法"
- 标注：此方法名称由提炼者命名，李开元未明确命名

### Q5: 提炼的方法如何验证有效性？

**验证方法**：
1. **内部验证**：在原著的其他案例中测试
2. **外部验证**：应用于其他研究对象
3. **同行评议**：请专家评估方法的合理性
4. **实践检验**：在实际研究中应用

**李开元案例**：
- 内部验证：10个案例成功应用
- 外部验证：可应用于《史记》其他章节
- 学术评价：李开元方法在学界有较高认可度

### Q6: 如何处理作者没有明确方法论章节的书？（新增）

**挑战**：
- 《安史之乱》没有专门的方法论章节
- 方法隐含在序言、章节标题、具体分析中
- 比李开元的《秦谜》（明确列出"七法"）更难提炼

**应对策略**：
1. **重点分析序言/前言/缘起**：
   - 作者往往在序言中阐述写作目标和立场
   - 《安史之乱》的"缘起"部分是方法论的浓缩
2. **关注副标题和章节标题**：
   - 副标题"历史、宣传与神话"直接揭示核心方法
   - 章节标题（如"'不惜'与'代价'的矛盾"）隐含方法论视角
3. **分析作者的自我定位**：
   - 作者与谁对比？（如vs《资治通鉴》、vs传统通俗读物）
   - 作者声称的创新是什么？（如"接近西方学术"、"财政核算"）
4. **利用推荐序和书评**：
   - 专家评价往往直接点明方法特征
   - 如罗振宇、陆大鹏的评语明确指出方法创新
5. **从具体案例反推方法**：
   - 看作者反复使用什么分析套路
   - 如多次使用财政计算 → 提炼出"后勤与财政计算法"

**实践验证**：
- 《安史之乱》虽无明确方法论章节，但通过上述策略成功提炼出8种方法
- 证明即使方法隐性，也可系统提炼

### Q7: 如何命名提炼出的方法？（新增）

**命名原则**：
1. **优先使用作者命名**（如果有）：
   - 李开元："排除法"、"动机分析法"（作者明确命名）
2. **借鉴学术术语**（如果作者未命名但有学术基础）：
   - "家族史料批判法"（借鉴"史料批判"术语）
3. **提炼者自创命名**（如果以上都没有）：
   - 命名要简洁（3-8个字）
   - 命名要准确（概括方法本质）
   - 命名要易懂（避免生僻术语）

**《安史之乱》命名实践**：
- **作者明确说明的**："宣传话语解构"（"摈弃宣传"）、"决策者视角复盘"（"复盘棋手的考量"）
- **借鉴学术术语的**："社会基础分析法"（引用仇鹿鸣"社会基础"概念）
- **提炼者命名的**："后勤与财政计算法"（作者做了但未命名）、"制度与人性互动分析法"

**命名注意事项**：
- 必须在文档中注明方法来源（作者原创/借鉴/提炼者命名）
- 提炼者命名的方法可能需要调整（根据后续研究）
- 避免过度抽象（如"系统分析法"太宽泛）

### Q8: 不同历史时期的方法能否互相借鉴？（新增）

**问题背景**：
- 李开元研究先秦秦汉（前597-前180）
- 《安史之乱》研究唐代（755-763）
- 时间跨越约1000年，方法能否通用？

**验证结果**：
1. **基础逻辑方法可通用**：
   - 排除法、动机分析法、反常检测法等在任何时期都适用
   - 这些方法基于人性和逻辑，不受时代限制
2. **依赖特定史料的方法需调整**：
   - 李开元的"出土史料互证法"：唐代出土史料较少，应用受限
   - 《安史之乱》的"财政计算法"：秦汉财政数据不如唐代详细，但仍可应用
3. **方法创新的时代特点**：
   - 秦汉史：史料相对简略，更需要推理和排除（李开元的"排除法"、"合理推想"）
   - 唐代史：史料相对丰富，更可能量化分析（《安史之乱》的"财政计算"）
4. **跨时代应用的价值**：
   - 李开元的"多源拼图法"可用于唐代（如安史之乱有唐廷vs叛军两个视角）
   - 《安史之乱》的"财政计算法"可用于楚汉战争（如刘邦vs项羽的经济基础对比）

**结论**：
- 大部分方法可跨时代应用
- 但需根据史料特点调整
- 不同时期的方法可以互补

## 七、进阶技巧

### 7.1 方法创新识别

**如何识别真正的方法创新**：
1. **与传统方法对比**：作者明确批判或超越传统方法
2. **跨学科借鉴**：引入其他学科的方法（如法律、医学）
3. **理论突破**：在元层次上提出新的理论框架
4. **操作创新**：设计新的操作流程或工具

**示例（李开元）**：
- 传统史学："有一分材料说一分话"，遇史料不足则存疑
- 李开元创新："一切历史都是推想"，提倡合理推想填补空白
- 理论突破：史真-史料-史实-史著四层次框架

### 7.2 方法间关系分析

**关系类型**：
1. **继承关系**：后作继承前作的方法
2. **深化关系**：后作深化前作的方法（更精细/更系统）
3. **组合关系**：多个基础方法组合成复合方法
4. **替代关系**：新方法替代旧方法

**分析工具**：
- 方法演进图谱
- 方法关系矩阵
- 方法应用频次统计

### 7.3 方法的机器化潜力评估

**可机器化的方法**：
- 有明确规则的方法（如法律鉴定、时间线重建）
- 可形式化的逻辑推理（如排除法）
- 可量化的分析（如数量验证）

**难以机器化的方法**：
- 依赖深度人文理解的方法（如性格一致性检验）
- 需要创造性洞察的方法（如反常检测）
- 依赖临场体验的方法

**AI辅助潜力**：
- LLM可辅助：动机分析、史料对比、逻辑推理
- LLM难以替代：历史判断、价值权衡、创新洞察
