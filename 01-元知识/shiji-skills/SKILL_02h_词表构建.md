# SKILL 02h: 词表构建(Glossary Building) — 专项词汇识别与结构化

> 从标注文本中提取特定领域的词汇表(glossary)，包括成语典故、专有名词、术语等，构建结构化词表并定位到原文段落。

---

## 一、任务定义

词表(glossary)构建是从史记全文中系统地提取和组织特定类型的词汇，包括：

1. **成语典故** — 成语、典故、经典名句的识别、定位与释义
2. **专有名词** — 官职、地名、族群、制度等封闭词表(controlled vocabulary)
3. **术语体系** — 礼制、历法、度量衡等专业术语
4. **文化词汇** — 器物、服饰、音乐、祭祀等文化概念

与词法分析（02e）的区别：
- **02e 词法分析**：字词级分类、词性标注、统计分析（面向语言学）
- **02h 词表构建(Glossary Building)**：领域词汇提取、语义标注、知识组织（面向知识图谱）

## 九、成语典故子任务详细说明

### 9.1 数据清洗流程

**原始数据问题**：
1. 括号概括 — 9条（如"（萧何定法，曹参守而勿失）"）
2. 非史记来源 — 2条（"狡兔三窟"出自《战国策》）
3. 章节标注错误 — 7条（如"唇亡齿寒"记为043实为039）

**清洗策略**：
```python
# 1. 排除括号概括
if original.startswith('（') and original.endswith('）'):
    remove_item()

# 2. 排除非史记来源
if verify_source() == "战国策":
    remove_item()

# 3. 修正章节错误
apply_manual_fixes({
    "唇亡齿寒": ("039", "晋世家", "30.33"),
    "过犹不及": ("067", "仲尼弟子列传", "52"),
    ...
})
```

### 9.2 模糊匹配算法参数

**相似度阈值**：
- 短语（≤5字）：80%
- 长文（>5字）：70%

**滑动窗口**：
- 窗口大小 = 模式长度 × (0.7 ~ 1.3)
- 步长 = 1字符

**特殊处理**：
- 省略号拆分：`text.split('…')[0]` 取首段匹配
- 实体标注穿透：`〖@人名〗` → `人名`

### 9.3 关键词高亮算法

**提取规则**：
```python
# 1. 拆分复合成语（如"浑沌/穷奇/梼杌/饕餮"）
if '/' in chengyu:
    keywords = chengyu.split('/')

# 2. 提取2-4字关键词
elif len(chengyu) >= 4:
    keywords = [chengyu[:2], chengyu[2:4], chengyu]

# 3. 按长度排序（长的优先匹配）
keywords.sort(key=len, reverse=True)
```

**高亮样式**：
```css
.item-original strong {
    color: #8b4513;           /* 棕色 */
    font-weight: 700;         /* 粗体 */
    text-decoration: underline;
    text-decoration-color: #d4a373;
    text-decoration-thickness: 2px;
}
```
