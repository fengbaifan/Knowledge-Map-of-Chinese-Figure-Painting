# SKILL 07e: 真实性推理 — 可信度评估与真伪判断

> 在史源溯源（SKILL_07d）的基础上，运用多维度推理方法评估《史记》记载的可信度和真实性。

---

## 一、核心问题

**这个记载是真是假？**

在已知史源类型和传播链的前提下，进一步判断：
- 这个史源本身是否可信？
- 记载内容是否符合逻辑？
- 是否存在矛盾和反常？
- 谁是受益者？谁有动机伪造？
- 综合判断：这是真相还是谣言？

**本SKILL的界限**：
- ✅ **关注**：史源可信度、记载真实性、逻辑自洽性、动机分析
- ⚠️ **前提**：需要先使用SKILL_07d完成史源溯源
- ❌ **不关注**：史源是什么、传播链如何（已由SKILL_07d完成）

**与SKILL_07c的关系**：
- **SKILL_07c (反常推理)**：异常检测器，批量扫描识别"这很奇怪"的记载，输出异常清单
- **SKILL_07e (真实性推理)**：深度分析器，针对重大疑案判断"这是真是假"，输出可信度评级
- **工作流**：07c可为07e提供研究线索，07e也可独立使用（用户直接提问）

## 二、可信度评级体系

### 2.1 A-E五级评分

| 等级 | 定义 | 史源+真实性 | 典型特征 |
|------|------|-----------|---------|
| **A** | 高度可信 | 明确档案+多源验证+逻辑自洽 | 官方档案、亲历者口述、无矛盾 |
| **B** | 较为可信 | 家传口述+时间近+逻辑合理 | 传承链清晰、符合常理、核心事实可信 |
| **C** | 存疑 | 时间远或细节可疑+部分逻辑通 | 核心事实可能可信，细节存疑 |
| **D** | 低可信度 | 来源不明+逻辑矛盾+动机可疑 | 违背常理、受益者存疑、可能是谣言 |
| **E** | 几乎肯定伪史 | 科学/法律/医学鉴定不通 | 明显违背客观规律、政治宣传 |

### 2.2 分层评估原则

**重要原则**：同一事件的不同层面可信度可能不同。

```
事件: 鸿门宴
  ├─ 核心事实（鸿门宴发生）: A级
  ├─ 基本经过（樊哙闯帐）: B级
  └─ 具体对话细节: C-D级
```

**评估维度**：
- **核心事实**：事件是否发生
- **基本经过**：主要人物行动
- **具体细节**：对话、神态、心理描写

## 三、七大推理方法

本方法论整合传统史源批判与李开元《秦谜》系列的推理方法。

### 方法1：医学/科学鉴定法

**定义**：应用基本医学和科学常识，验证历史记载是否符合客观规律。

**常用知识库**：
- 生理常识：怀孕周期、生育年龄
- 时间常识：距离与速度、工程耗时
- 数量常识：人口容量、军队规模

**应用步骤**：
```
1. 识别可验证的科学要素
2. 调查古代医学知识水平
3. 应用基本科学常识
4. 检验是否存在科学矛盾
5. 判断记载的真实性
```

**案例：秦始皇身世的医学鉴定**

```
《史记》记载:
085_吕不韦列传: "姬自匿有身，至大期时，生子政"

医学分析:
  - "大期"= 十个月（足月怀孕）
  - 如果吕不韦献姬时已孕2月 → 应8月生产（早产）
  - 但嬴政"大期时生"→ 说明是足月生产
  - 从子楚娶姬算起，正好十个月

结论:
  医学证据支持嬴政是子楚之子
  "吕不韦献有孕之姬"的说法存在医学矛盾
```

**可机器化**：可建立科学常识库，中高度可机器化(★★★★☆)

---

### 方法2：法律/制度鉴定法

**定义**：运用古代法律制度知识，检验历史事件是否符合法定程序。

**检验维度**：
1. **法律程序**：是否符合当时法律规定？
2. **违法成本**：违法的后果是什么？
3. **制度约束**：制度能否防止该行为？

**应用步骤**：
```
1. 查明当时的法律制度
2. 对照史料与法律规定
3. 检验是否符合法定程序
4. 判断违法的可能性和代价
5. 推断记载的真实性
```

**案例：秦国继承法鉴定**

| 法律要素 | 秦国制度 | 献有孕之姬的合法性 |
|---------|---------|------------------|
| 血统纯正性 | 极为重要（宗法制度） | ❌ 违背 |
| 继承程序 | "刻玉符"严格登记 | ❌ 无法通过 |
| 违法后果 | 废除继承权、杀身之祸 | ❌ 风险极高 |
| 商人理性 | 吕不韦的投资目的 | ❌ 不符合商业逻辑 |

**结论**：
- 献有孕之姬在法律上不可能成立
- 违背吕不韦的商业理性
- 该说法可信度：D-E级（几乎肯定是谣言）

**可机器化**：可建立法律规则引擎，高度可机器化(★★★★★)

---

### 方法3：动机分析法

**定义**：分析当事人的利益诉求，判断行为是否符合"趋利避害"的人性规律。

**核心原则**：
- 人的行为符合利益最大化
- 寻找"谁是受益者"
- 受益者最可能是行为主体或谣言制造者

**分析框架**：
```
FOR 每个相关人物 P:
  计算行为A对P的利益影响
    - 短期利益
    - 长期利益
    - 风险收益比
  判断: A是否符合P的"趋利避害"？
    IF 不符合 THEN 存疑
    IF 符合 THEN P可能是主体/受益者
```

**案例：吕不韦献姬传说的动机分析**

**假设A：传说为真（嬴政确实是吕不韦之子）**

| 相关人物 | 动机分析 | 符合利益吗？ |
|---------|---------|------------|
| 吕不韦 | 泄露真相 → 自毁前程、杀身之祸 | ❌ 严重违背 |
| 子楚 | 真相曝光 → 颜面尽失、继承权受质疑 | ❌ 严重违背 |
| 嬴政 | 真相传播 → 帝位正统性受损 | ❌ 严重违背 |

**假设B：传说为假（嬴政是子楚之子，但吕不韦传播假消息）**

| 相关人物 | 动机分析 | 符合利益吗？ |
|---------|---------|------------|
| 吕不韦 | 传播假消息 → 巩固地位（"皇帝是我儿子"） | ✅ 符合政治利益 |
| 反秦势力 | 传播谣言 → 质疑秦始皇正统性 | ✅ 符合政治目的 |
| 司马迁 | 记录传闻 → 展示秦朝权力黑幕 | ✅ 符合史家立场 |

**结论**：
- 假设B更符合各方利益
- 吕不韦有动机传播假消息（即使嬴政不是他儿子）
- 反秦势力有动机传播谣言
- 该说法可信度：D级（政治谣言）

**可机器化**：可建立利益关系图谱，中度可机器化(★★★☆☆)

### 方法4：逻辑自洽性检验

**定义**：检验史书记载是否符合内部逻辑一致性，识别前后矛盾、因果断裂等不自洽现象。

**检验维度**：
1. **时间逻辑**：事件时间线是否连贯，无矛盾
2. **因果逻辑**：原因→结果链条是否完整，无断裂
3. **人物行为逻辑**：行为是否符合人物性格/利益/能力
4. **叙事逻辑**：细节是否过于完美/戏剧化/不合理

**分析步骤**：
```
1. 检验时间线连贯性（前后事件是否矛盾）
2. 检验因果链完整性（A→B→C是否成立）
3. 检验人物行为合理性（动机/能力/性格是否匹配）
4. 识别叙事异常点（文学加工痕迹）
5. 判断记载的逻辑自洽程度
```

**与SKILL_07c的区别**：
- **07c反常推理**：关注"违反常识/制度的单条记载"（白起坑杀40万）
- **07e逻辑自洽性**：关注"记载内部的逻辑一致性"（秦始皇身世前后矛盾）

**案例：秦始皇身世的逻辑自洽性检验**

**不自洽之一：赵姬隐瞒怀孕的逻辑矛盾**

```
常规逻辑: 怀孕10个月无法隐瞒
记载: "姬自匿有身"

反常分析:
  可能原因1: 记载有误
  可能原因2: 只隐瞒了最初1-2个月
  可能原因3: 完全虚构

判断: 如果只隐瞒1-2个月，逻辑上可行
```

**不自洽之二：至大期时生的时间矛盾**

```
如果吕不韦献姬时已孕2月:
  正常情况 → 8个月生产（早产）
  记载 → "大期时生"（足月）

反常分析:
  可能原因1: 秦朝官方篡改记录（故意写"大期"）
  可能原因2: 确实是足月，说明是子楚之子

判断: 存在两种对立解释，无法确定
```

**不自洽之三：秦国官方沉默的因果矛盾**

```
常规逻辑: 如果真相是吕不韦之子，秦国官方应该澄清
记载: 秦国官方始终沉默

反常分析:
  可能原因: 根本没有这回事，不值得澄清

判断: 沉默本身可能说明是谣言
```

**可机器化**：需建立逻辑规则库+因果链模型，中度可机器化(★★★☆☆)

### 方法5：沉默证据法

**定义**：应有而未有的记载本身就是证据，分析沉默的原因。

**沉默类型**：
1. **偶然失载**：史料散佚、记录遗漏
2. **刻意隐瞒**：政治忌讳、家族丑闻
3. **本来就不存在**：伪史、谣言

**分析步骤**：
```
1. 识别"应有而未有"的记载
2. 列出应该存在的证据清单
3. 分析沉默的原因:
   - 偶然失载？（时间久远）
   - 刻意隐瞒？（政治原因）
   - 本来就不存在？（伪史）
4. 从沉默推断真实性
```

**案例：秦始皇身世的沉默证据**

**应有的证据清单**：

```
如果"吕不韦献有孕之姬"为真，应该存在:
  ✗ 秦国官方承认或澄清
  ✗ 子楚的反应记载
  ✗ 嬴政的回应
  ✗ 吕不韦被诛时的此项罪名
  ✗ 秦国宗室的质疑记录
```

**沉默分析**：

| 应有证据 | 是否存在 | 沉默原因 |
|---------|---------|---------|
| 秦国官方态度 | ✗ 无记载 | 本来就不存在此事？ |
| 子楚反应 | ✗ 无记载 | 本来就不存在此事？ |
| 嬴政回应 | ✗ 无记载 | 可能不屑回应谣言？ |
| 吕不韦被诛罪名 | ✗ 无此罪 | 说明不是真相 |

**结论**：
- 关键证据全部缺失
- 沉默本身说明可能是谣言
- 该说法可信度：D级

**可机器化**：需要完整性知识图谱，低可机器化(★★☆☆☆)

### 方法6：排除法

**定义**：当嫌疑对象有限时，逐一排除不符合逻辑的选项，剩下的即为最可能的答案。

**操作步骤**：
```
1. 列出所有可能的解释/嫌疑对象
2. 分析每个选项的动机和条件
3. 应用其他方法（医学、法律、动机等）逐一检验
4. 排除不符合逻辑的选项
5. 剩下的即为最可能的答案
```

**案例：秦始皇生父之谜**

**假设列表**：

```
解释A: 嬴政是子楚之子
解释B: 嬴政是吕不韦之子
```

**逐一检验**：

| 检验方法 | 解释A（子楚之子） | 解释B（吕不韦之子） |
|---------|----------------|------------------|
| 医学鉴定 | ✅ "大期时生"符合 | ❌ "大期"矛盾 |
| 法律鉴定 | ✅ 符合继承法 | ❌ 违背继承法 |
| 动机分析 | ✅ 各方符合利益 | ❌ 吕不韦违背利益 |
| 沉默证据 | ✅ 无需额外证据 | ❌ 关键证据缺失 |
| 官方记载 | ✅ "庄襄王子也" | ❌ 秦国从未承认 |

**排除过程**：
- 解释B违背医学、法律、动机、沉默证据
- 解释A符合所有检验

**结论**：
- 排除解释B
- 解释A（子楚之子）最可能
- 但承认不确定性：真相不可知

**可机器化**：可建立决策树模型，中高度可机器化(★★★★☆)

### 方法7：临场体验法

**定义**：设身处地想象历史情境，用人性常理判断事件合理性。

**应用维度**：
1. **空间体验**：实地考察遗址，感受距离/地形
2. **时间体验**：计算时间跨度，验证可行性
3. **心理体验**：换位思考，判断人物心理
4. **社会体验**：理解当时社会规范和人情常理

**方法特点**：
- 依赖人类共情能力
- 难以完全机器化
- 需要历史想象力
- 结合考古实地考察

**案例：沙丘之谋的临场体验**

**问题**：扶苏为何不验证诏书真伪？

```
临场体验:
  1. 心理: 扶苏性格（仁孝、谨慎）
  2. 社会: 秦朝君父权威（不敢质疑）
  3. 时间: 赵高可能限定时间（紧迫感）
  4. 空间: 扶苏在边疆，信息不对称

人性判断:
  - 孝道压力: 不敢质疑"父皇诏书"
  - 性格因素: 扶苏过于仁孝
  - 蒙恬已劝阻，但扶苏不听

可能性分析:
  选项1: 扶苏过于孝顺，不敢质疑（较可信）
  选项2: 赵高伪造技术高超（可能）
  选项3: 记载有误，实际被迫自杀（存疑）
```

**结论**：
- 从人性角度，扶苏不验证是可能的
- 但仍存在记载失真的可能
- 需结合其他证据综合判断

**可机器化**：低，依赖人类共情和想象力(★☆☆☆☆)

## 四、综合评估流程

### 4.1 完整分析流程

```
【前置步骤：SKILL_07d史源溯源】
  ↓ [输出：史源类型、传播链、传播层级]

【第一层：客观规律检验】
  ↓
1. 医学/科学鉴定
   - 检查生理常识
   - 验证时间/数量合理性
   - 识别科学矛盾
  ↓
2. 法律/制度鉴定
   - 查明当时法律
   - 检查程序合法性
   - 评估违法成本
  ↓

【第二层：人性逻辑检验】
  ↓
3. 动机分析
   - 识别相关人物
   - 分析利益关系
   - 判断"谁是受益者"
  ↓
4. 逻辑自洽性检验
   - 检验时间线连贯性
   - 检验因果链完整性
   - 检验人物行为合理性
  ↓
5. 沉默证据分析
   - 列出应有证据
   - 分析沉默原因
   - 推断隐藏事实
  ↓
6. 临场体验
   - 换位思考
   - 人性判断
   - 常理验证
  ↓

【第三层：综合判断】
  ↓
7. 排除法
   - 汇总所有证据
   - 逐一排除假象
   - 得出最可能结论
  ↓

【最终输出】
  ↓
8. 可信度评级（A/B/C/D/E）
9. 分层评估（核心事实 vs 细节）
10. 多种解释（承认不确定性）
```

### 4.2 评级决策矩阵

**A级（高度可信）**：
- ✅ 史源：档案文献或亲历者口述
- ✅ 传播链：2层以内
- ✅ 科学/法律检验通过
- ✅ 动机分析合理
- ✅ 无明显矛盾

**B级（较为可信）**：
- ✅ 史源：家传口述
- ✅ 传播链：2-3层
- ✅ 核心事实逻辑自洽
- ⚠️ 细节可能有文学加工

**C级（存疑）**：
- ⚠️ 史源：二手传闻
- ⚠️ 传播链：3-5层
- ⚠️ 部分逻辑通，部分存疑
- ⚠️ 核心事实可能可信，细节存疑

**D级（低可信度）**：
- ❌ 史源：来源不明
- ❌ 逻辑矛盾
- ❌ 动机可疑
- ❌ 可能是谣言或政治宣传

**E级（几乎肯定伪史）**：
- ❌ 医学/科学/法律鉴定明确不通
- ❌ 明显违背客观规律
- ❌ 政治宣传或神话色彩

## 五、典型案例对比

### 案例01：鸿门宴

**详细分析见**：[`labs/source-inference/source_inference_案例01_鸿门宴.md`](../labs/source-inference/source_inference_案例01_鸿门宴.md)

**史源溯源（SKILL_07d）**：
- 史源类型：家传口述（樊哙家族）
- 传播链：樊哙 → 樊市人/樊他广 → 司马谈/司马迁
- 传播层级：2-3层

**真实性推理（SKILL_07e）**：

| 推理方法 | 分析结果 |
|---------|---------|
| 动机分析 | ✅ 樊哙家族有传承动机，符合刘邦阵营利益 |
| 反常检测 | ✅ 无明显矛盾 |
| 临场体验 | ✅ 樊哙性格与行为吻合 |

**分层评估**：
