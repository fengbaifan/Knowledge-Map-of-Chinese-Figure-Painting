## 设计理念

### 1. 为什么标注动词

动词是历史叙事的核心要素，标注动词能够：

- **事件提取**：自动识别军事行动（伐/攻/击）、刑罚事件（杀/诛/斩）等历史事件
- **关系挖掘**：建立人物-动作-对象的三元组关系（如：秦伐楚、王诛臣）
- **语义检索**：按动词类型检索特定事件（如：所有"伐"的记录）
- **统计分析**：量化分析军事冲突频率、刑罚执行模式等历史规律

### 2. 动词与名词的区分

| 维度 | 名词实体 | 动词实体 |
|------|---------|---------|
| **外层符号** | `〖TYPE内容〗` (U+3016/3017 白色龟甲括号) | `⟦TYPE动词⟧` (U+27E6/27E7 数学双方括号) |
| **标注对象** | 静态实体（人名/地名/官职等） | 动态行为（军事/刑罚/政治等） |
| **TYPE符号** | 文本符号（@=;%&^等） | 几何符号（◈◉○◇） |
| **语义功能** | 事件参与者（主语/宾语） | 事件核心（谓语） |
| **示例** | 〖◆楚〗〖@子常〗 | ⟦◈伐⟧⟦◉杀⟧ |

**核心原则**：使用不同的外层符号，确保动词与名词在视觉和语义上明确区分。

## 标注语法

### 1. 基本格式

```
⟦TYPE动词⟧
```

- **外层符号**：`⟦⟧` (U+27E6 MATHEMATICAL LEFT WHITE SQUARE BRACKET / U+27E7 MATHEMATICAL RIGHT WHITE SQUARE BRACKET)
- **TYPE**：动词类型符号（见下文分类体系）
- **动词**：单字或多字动词

### 2. 消歧格式

```
⟦TYPE动词|消歧说明⟧
```

- **消歧说明**：用于明确歧义词的具体含义
- **分隔符**：管道符 `|`（U+007C）

**消歧示例**：

```markdown
⟦◈取⟧〖=六〗、〖=潜〗               # 基本标注
⟦◈取|攻取⟧〖=六〗、〖=潜〗          # 消歧说明：攻取城邑

⟦◈走⟧〖◆郑〗                       # 基本标注
⟦◈走|逃走⟧〖◆郑〗                  # 消歧说明：逃亡之走，非行走之走

⟦◉死⟧罪                           # 基本标注
⟦◉死|处死⟧罪                      # 消歧说明：判处死刑，非自然死亡
```

### 3. 语法约束

**✅ 正确示例**：

```markdown
⟦◈伐⟧〖◆随〗                       # 单字动词
⟦◉诛⟧〖@重黎〗而不尽               # 带宾语
⟦◈击|攻击⟧〖◆楚〗于〖=城濮〗       # 消歧+地点
〖◆吴〗⟦◈伐⟧⟦◈败⟧〖@子常〗         # 连续动词
```

**❌ 错误示例**：

```markdown
⟦⟦◈伐⟧⟧                          # 禁止嵌套
⟦◈〗                              # 禁止空标注
⟦◈伐|⟧                            # 消歧说明不能为空
〖◈伐〗                            # 错误的外层符号（应用⟦⟧）
⟦@伐⟧                             # 错误的TYPE符号（◈非@）
```

### 3. 刑罚制度名词（保留 `〖[legal〗`）

**重要区分**：刑罚动词 vs 刑罚制度名词

| 类别 | 标注格式 | 特征 | 示例 |

## 标注原则

### 1. 动词识别原则

**✅ 应标注的动词**：
- 历史事件的核心动作（军事/刑罚/政治等）
- 具有明确主语和宾语的及物动词
- 可用于事件提取和关系挖掘的关键动词

**❌ 不应标注的动词**：
- 一般性日常动词（如：行、走、坐、立、言、语）
- 助动词和连系动词（如：能、可、是、为）
- 趋向动词（如：来、去、上、下，除非表示军事机动）

### 2. 消歧原则

**需要消歧的情况**：

1. **同形异义**：一个字有多种动词含义
   ```markdown
   ⟦◈取|攻取⟧〖=城〗      # 军事之取
   取而代之                # 一般之取（不标注）

   ⟦◈走|逃走⟧奔〖◆郑〗    # 逃亡之走
   走马观花                # 行走之走（不标注）
   ```

2. **跨类歧义**：同一字可属于多个动词类型
   ```markdown
   ⟦◈亡|逃亡⟧奔〖◆郑〗    # 军事逃亡
   ⟦◉亡|逃罪⟧出奔        # 逃避刑罚
   ```

3. **与名词混淆**：避免与同形名词混淆
   ```markdown
   ⟦◈围⟧〖=邯郸〗         # 动词：包围
   解〖=邯郸〗之围         # 名词：包围圈（不标注）
   ```

### 3. 标注边界原则

**单字标注**：大部分动词为单字
```markdown
⟦◈伐⟧、⟦◈攻⟧、⟦◈击⟧、⟦◉杀⟧、⟦◉诛⟧
```

**多字标注**：复合动词可标注为整体（谨慎使用）
```markdown
⟦◈击破⟧〖◆楚〗          # 击破作为一个整体
⟦◈击⟧⟦◈破⟧〖◆楚〗      # 或拆分为两个动词（推荐）
```

### 4. 优先级原则

当一个词同时满足多个类型时，按以下优先级标注：

1. **军事动词** `⟦◈⟧` 优先于刑罚动词
   ```markdown
   ⟦◈杀⟧敌千人           # 战场杀敌用军事标注
   ⟦◉杀⟧〖@大臣〗        # 政治处决用刑罚标注
   ```

2. **刑罚动词** `⟦◉⟧` 优先于制度名词
   ```markdown
   ⟦◉斩⟧〖@颜良〗        # 动作：斩杀
   〖[斩首〗八万           # 名词：斩首刑
   ```

3. **特定上下文优先**：根据语境判断
   ```markdown
   〖◆晋〗⟦◈救⟧〖◆宋〗    # 军事救援
   救民于水火             # 一般救助（不标注）
   ```

## 与名词实体的区别

### 1. 符号对比

| 对比项 | 名词实体 | 动词实体 |
|-------|---------|---------|
| **外层符号** | `〖〗` 白色龟甲括号 | `⟦⟧` 数学双方括号 |
| **Unicode** | U+3016 / U+3017 | U+27E6 / U+27E7 |
| **TYPE符号类型** | 文本符号（@=;%等） | 几何符号（◈◉○◇） |
| **视觉区分** | 较圆润的括号 | 较方正的括号 |

### 2. 语义对比

| 对比项 | 名词实体 | 动词实体 |
|-------|---------|---------|
| **标注对象** | 静态实体（人、地、物） | 动态行为（动作、事件） |
| **句法功能** | 主语、宾语、定语 | 谓语 |
| **语义角色** | 事件参与者 | 事件核心 |
| **持久性** | 持续存在 | 瞬时发生 |

### 3. 混合标注示例

```markdown
〖◆楚〗⟦◈伐⟧〖◆随〗
 名词  动词  名词
 主语  谓语  宾语

〖@帝喾〗以〖%庚寅〗日⟦◉诛⟧〖@重黎〗
  主语    时间状语   动词    宾语

〖◆吴〗⟦◈伐⟧⟦◈败⟧〖@子常〗，〖@子常〗⟦◉亡⟧奔〖◆郑〗
 主语  动词1 动词2  宾语      主语    动词  趋向 宾语
```

## 标注示例

### 1. 军事动词示例

#### 示例1：单纯军事行动

**原文**：
```
楚伐随。楚灭邓。
```

**标注**：
```markdown
〖◆楚〗⟦◈伐⟧〖◆随〗。〖◆楚〗⟦◈灭⟧〖◆邓〗。
```

**解析**：
- `⟦◈伐⟧`：征伐，进攻类军事动词
- `⟦◈灭⟧`：消灭，交战类军事动词

#### 示例2：复杂战役叙述

**原文**：
```
吴伐败子常，子常亡奔郑
```

**标注**：
```markdown
〖◆吴〗⟦◈伐|征伐⟧⟦◈败|击败⟧〖@子常〗，〖@子常〗⟦◈亡|逃亡⟧奔〖◆郑〗
```

**解析**：
- `⟦◈伐|征伐⟧`：征伐，添加消歧说明
- `⟦◈败|击败⟧`：击败，明确为主动击败
- `⟦◈亡|逃亡⟧`：逃亡，区分于"死亡"之亡

#### 示例3：攻城战术

**原文**：
```
攻宛、舞阳，下城七十余，破秦军，斩首八万
```

**标注**：
```markdown
⟦◈攻⟧〖=宛〗、〖=舞阳〗，⟦◈下⟧〖=城〗七十余，⟦◈破⟧〖*秦军〗，〖[斩首〗八万
```

**解析**：
- `⟦◈攻⟧`：进攻城池
- `⟦◈下⟧`：攻下，占领
- `⟦◈破⟧`：击破军队
- `〖[斩首〗`：制度名词（非动词），表示斩首刑的执行结果

### 2. 刑罚动词示例

#### 示例1：政治处决

**原文**：
```
帝喾使重黎诛之而不尽。帝乃以庚寅日诛重黎
```

**标注**：
```markdown
〖@帝喾〗使〖@重黎〗⟦◉诛⟧之而不尽。帝乃以〖%庚寅〗日⟦◉诛⟧〖@重黎〗
```

**解析**：
- `⟦◉诛⟧`：诛杀，刑罚动词
- 两次出现，分别为对不同对象的诛杀

#### 示例2：弑君事件

**原文**：
```
商臣以宫兵围成王，成王请食熊蹯而死，商臣弑成王
```

**标注**：
```markdown
〖@商臣〗以宫兵⟦◈围|围困⟧〖@成王〗，〖@成王〗请食〖+熊蹯〗而⟦◉死|处死⟧，〖@商臣〗⟦◉弑|弑君⟧〖@成王〗
```

**解析**：
- `⟦◈围|围困⟧`：用宫兵包围，军事动词（虽用于政变，但行为本身是军事性的）
- `⟦◉死|处死⟧`：被迫处死
- `⟦◉弑|弑君⟧`：弑君，添加消歧说明明确性质

#### 示例3：刑罚制度混用

**原文**：
```
杀令尹子西，斩首八万。当死，赎为城旦
```

**标注**：
```markdown
⟦◉杀⟧〖;令尹〗〖@子西〗，〖[斩首〗八万。当⟦◉死⟧，〖[赎为〗〖[城旦〗
```

**解析**：
- `⟦◉杀⟧`：动词，杀害大臣
- `〖[斩首〗`：制度名词，斩首刑
- `⟦◉死⟧`：动词，判处死刑
- `〖[赎为〗`、`〖[城旦〗`：制度名词，刑罚制度术语

### 3. 军事+刑罚混合示例

#### 示例1：战争与处罚

**原文**：

**解析**：

## 质量检查规范

### 1. 核心原则

**所有已经标注的名词内部不需要作动词标注**

这是动词标注的最重要原则：

- ✅ **正确**：名词实体完整保留，不拆分标注动词
  ```markdown
  〖◆战国〗                    # 战国是朝代名，完整标注
  〖@破奴〗                    # 破奴是人名，完整标注
  〖;游击将军〗                # 游击将军是官职，完整标注
  〖=取虑〗                    # 取虑是地名，完整标注
  ```

- ❌ **错误**：在名词实体内部嵌套标注动词
  ```markdown
  〖◆⟦◈战⟧国〗                # 错误！战国被拆分
  〖@⟦◈破⟧奴〗                # 错误！破奴被拆分
  〖;游⟦◈击⟧将军〗            # 错误！游击将军被拆分
  〖=⟦◈取⟧虑〗                # 错误！取虑被拆分
  ```

**原因**：
1. **语义完整性**：破奴、战国、游击将军等是完整的专有名词，不应拆解
2. **上下文决定性**：同一字符在不同上下文有不同词性，名词内部的字不是独立动词
3. **标注冲突**：嵌套标注会导致解析歧义和显示错误

### 2. 嵌套检查规则

#### 规则1：禁止动词标注内嵌套实体标注

**错误模式**：`⟦TYPE...〖...〗...⟧`

```markdown
# ❌ 错误示例
⟦◈伐〖◆楚〗国⟧              # 动词标注内包含实体标注

# ✅ 正确标注
⟦◈伐⟧〖◆楚国〗              # 动词和实体分开标注
```

#### 规则2：禁止实体标注内嵌套动词标注

**错误模式**：`〖TYPE...⟦...⟧...〗`

```markdown
# ❌ 错误示例
〖◆⟦◈战⟧国〗                # 实体标注内包含动词标注
〖@⟦◈破⟧奴〗                # 人名内包含动词
〖;游⟦◈击⟧将军〗            # 官职内包含动词
〖=⟦◈取⟧虑〗                # 地名内包含动词

# ✅ 正确标注
〖◆战国〗                    # 战国是完整朝代名
〖@破奴〗                    # 破奴是完整人名
〖;游击将军〗                # 游击将军是完整官职
〖=取虑〗                    # 取虑是完整地名
```

#### 规则3：禁止同类标注嵌套

**错误模式**：`⟦⟦...⟧⟧` 或 `〖〖...〗〗`

```markdown
# ❌ 错误示例
⟦◈⟦◈伐⟧⟧                  # 动词标注嵌套
〖◆〖◆楚〗〗                 # 实体标注嵌套

# ✅ 正确标注
⟦◈伐⟧                      # 单层动词标注
〖◆楚〗                     # 单层实体标注
```

### 3. 常见嵌套错误模式

#### 模式1：朝代/时期名称

```markdown
# ❌ 错误
〖◆⟦◈战⟧国〗                # 战国
〖◆⟦◈春⟧秋〗                # 春秋

# ✅ 正确
〖◆战国〗
〖◆春秋〗
```

#### 模式2：人名

```markdown
# ❌ 错误
〖@⟦◈破⟧奴〗                # 破奴(卫青将领)
〖@⟦◈破⟧石〗                # 破石(人名)
〖@⟦◈去⟧病〗                # 去病(霍去病)
〖@⟦◉杀⟧⟧青〗               # 杀青(人名)

# ✅ 正确
〖@破奴〗
〖@破石〗
〖@去病〗
〖@杀青〗
```

#### 模式3：官职名称

```markdown
# ❌ 错误
〖;游⟦◈击⟧将军〗            # 游击将军
〖;⟦◈破⟧敌将军〗            # 破敌将军
〖;⟦◉执⟧法官〗              # 执法官

# ✅ 正确
〖;游击将军〗
〖;破敌将军〗
〖;执法官〗
```

#### 模式4：地名

```markdown
# ❌ 错误
〖=⟦◈取⟧虑〗                # 取虑(古地名)
〖=⟦◈伐⟧檀〗                # 伐檀(地名)

# ✅ 正确
〖=取虑〗
〖=伐檀〗
```

#### 模式5：文献名称

```markdown
# ❌ 错误
〖{⟦◈伐⟧檀〗                # 《伐檀》诗经篇名
〖{⟦◉刺⟧客列传〗            # 《刺客列传》

# ✅ 正确
〖{伐檀〗
〖{刺客列传〗
```

### 4. 判断技巧

#### 技巧1：查字典/人名表

遇到疑似动词的字符时，先查阅：
- 人名词典（破奴、去病等常见人名）
- 地名词典（取虑等古地名）
- 官职表（游击将军等）

#### 技巧2：看上下文

```markdown
# 示例：战
〖◆战国〗时期                # "战国"是朝代，非动词"战"
与〖◆楚〗⟦◈战⟧于〖=泓〗    # 此处"战"是交战动词

# 示例：破
〖@破奴〗领兵                # "破奴"是人名，非动词"破"
⟦◈破⟧〖◆楚〗军              # 此处"破"是击破动词
```

#### 技巧3：检查标注完整性

如果已有实体标注，说明前人判断为专有名词：
```markdown
〖@破奴〗                    # 已有@标记，确认是人名
〖◆战国〗                    # 已有'标记，确认是朝代
```

不应再拆分其内部字符标注为动词。

### 5. 质量检查工具

#### 工具1：Lint检查器

**脚本**：`kg/entities/scripts/lint_verb_tagging.py`

**检查项目**：
1. **嵌套问题**（ERROR级别）
   - 动词标注内包含实体标注
   - 实体标注内包含动词标注
   - 同类标注嵌套
   - 交叉嵌套（括号不匹配）

2. **格式错误**（ERROR级别）
   - 未闭合的标注
   - 括号类型混用

### 8. 常见问题排查

#### Q1: 如何判断一个词是专有名词还是动词？

**排查步骤**：
1. 查阅史记人名索引（破奴、去病等常见人名）
2. 检查是否已有实体标注（如已标注〖@破奴〗，说明是人名）
3. 分析上下文（破奴领兵 vs 破敌千人）
4. 搜索该词在其他章节的用法

#### Q2: Lint检查报告嵌套错误，但我认为是动词，如何处理？

**处理方法**：
1. 首先确认实体标注是否正确（可能是实体标注错误）
2. 如果实体标注正确，必须移除内部动词标注（遵守核心原则）
3. 如果确认应该是动词，需要移除外层实体标注

**示例**：
```markdown
# 情况1：实体标注正确
〖@⟦◈破⟧奴〗 → 〖@破奴〗    # 移除动词标注，保留人名

# 情况2：实体标注错误
〖@破奴〗领兵⟦◈破⟧敌千人
         ↑这里的"破"是动词，但"破奴"是人名
修正：〖@破奴〗领兵⟦◈破⟧敌千人  # 正确！
```

#### Q3: 如何处理同形异义词（如"破"既是人名成分又是动词）？

**处理原则**：
- 上下文决定词性
- 已标注实体内部不再标注动词
- 同一字在不同位置可以有不同标注

**示例**：
```markdown
〖@破奴〗⟦◈破⟧〖◆匈奴〗
  人名     动词   国名
  ↑ "破"是人名一部分，不标注为动词
           ↑ "破"是击破动词，标注为⟦◈破⟧
```

## 常见问题

### Q1: 如何区分军事动词和刑罚动词？

**A**: 根据上下文判断：

- **军事动词**：国与国、军队与军队之间的战争行为
  ```markdown
  〖◆秦〗⟦◈伐⟧〖◆楚〗        # 国家间战争
  ⟦◈杀⟧敌千人              # 战场杀敌
  ```

- **刑罚动词**：政治内部的惩处、处决行为
  ```markdown
  ⟦◉杀⟧〖;丞相〗〖@李斯〗  # 政治处决
  ⟦◉诛⟧〖@奸臣〗           # 刑罚诛杀
  ```

### Q2: 多字动词如何标注？

**A**: 推荐拆分为单字动词：

```markdown
# 推荐：拆分标注
⟦◈击⟧⟦◈破⟧〖◆楚〗

# 可选：整体标注（谨慎使用）
⟦◈击破⟧〖◆楚〗
```

**理由**：单字标注便于统计分析和语义检索。

### Q3: 如何判断是否需要消歧？

**A**: 以下情况需要消歧：

1. 同形异义词（如：走/取/下）
2. 在文本中频繁出现且含义不一致
3. 对理解历史事件有重要影响

**不需要消歧**：含义明确、上下文清晰的词。

### Q4: 动词标注是否会影响阅读体验？

**A**: 标注后可通过CSS控制显示：

- **默认模式**：仅高亮显示，不影响阅读
- **标注模式**：显示完整标注符号
- **悬停提示**：鼠标悬停显示动词类型

### Q5: 如何处理"引兵""发兵""起兵"等多字军事词？

**A**: 这些属于**军事名词短语**，而非单纯动词：

```markdown
〖@项羽〗〖[引兵〗渡〖=河〗     # 制度/术语名词
〖@陈胜〗〖[起兵〗于〖=大泽〗   # 制度/术语名词
```

如未来扩展，可标注为军事动词：
```markdown
〖@项羽〗⟦◈引兵⟧渡〖=河〗     # 军事动词（未来）
```

### Q6: 旧标注格式如何迁移？

**A**: 使用迁移工具 `migrate_verb_tags.py`：

```bash
# 预览迁移
python kg/entities/scripts/migrate_verb_tags.py --chapter 040 --dry-run

# 执行迁移
python kg/entities/scripts/migrate_verb_tags.py --all --backup

# 迁移结果
〖[伐〗 → ⟦◈伐⟧
〖[杀〗 → ⟦◉杀⟧
〖[斩首〗 → 保持不变（制度名词）
```
