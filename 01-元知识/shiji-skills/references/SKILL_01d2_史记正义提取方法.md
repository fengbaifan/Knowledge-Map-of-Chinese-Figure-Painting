# SKILL_01d2 史记正义提取方法

> **参考文档**：从《史记正义》提取特殊读音的方法论、工具使用、质量检查流程

---

## 一、方法论概述

### 1.1 《史记正义》简介

**作者**：唐代张守节
**性质**：《史记》三家注之一，最权威的读音来源
**价值**：标注了大量古今读音差异的字词

**注音方法**：
- **反切**：中古汉语标准注音法（如"於，乌胡反"）
- **直音**：直接指出读音（如"音焉与"）
- **通假**：说明通假关系（如"驯读曰训"）

### 1.2 为什么需要提取

**原因**：
1. 古代地名、人名读音特殊
2. 现代拼音库缺少这些冷僻词
3. 《史记正义》是最权威的读音来源

**适用范围**：
- 古地名（如"会稽"kuài jī，非huì jī）
- 古人名（如"召公"shào gōng，非zhào gōng）
- 族群称号（如"冒顿"mò dú，非mào dùn）
- 特殊官职（如"仆射"pú yè，非pú shè）

## 二、读音注释模式识别

### 2.1 模式1：反切注音

**格式**：`某音甲乙反`

**含义**：取"甲"字的声母 + "乙"字的韵母和声调

**示例**：

| 原文 | 反切 | 现代拼音 | 说明 |
|------|------|---------|------|
| 阏 | 于达反 | yān | 取"于"(yú)声母y + "达"(dá)韵母an |
| 句 | 古侯反 | gōu | 取"古"(gǔ)声母g + "侯"(hóu)韵母ou |
| 亢 | 苦浪反 | kàng | 取"苦"(kǔ)声母k + "浪"(làng)韵母àng |

**反切转拼音规则**：

```python
def fanqie_to_pinyin(shengmu_char, yunmu_char):
    """
    shengmu_char: 反切上字（取声母）
    yunmu_char: 反切下字（取韵母+声调）
    """
    shengmu = get_initial(shengmu_char)  # 提取声母
    yunmu = get_final(yunmu_char)        # 提取韵母
    tone = get_tone(yunmu_char)          # 提取声调

    return shengmu + yunmu + tone
```

### 2.2 模式2：直接注音

**格式**：`音某` 或 `某音某`

**示例**：

| 原文 | 注音 | 现代拼音 |
|------|------|---------|
| 阏与 | 音焉与 | yān yǔ |
| 朝鲜 | 朝音潮 | cháo |
| 令支 | 令音铃 | líng |

**处理方法**：
- 直接查"音后字"的拼音
- 注意声调可能需要调整

### 2.3 模式3：通假注音

**格式**：`某读曰某` 或 `某读为某`

**示例**：

| 原文 | 注音 | 含义 |
|------|------|------|
| 驯 | 驯读曰训 | 通"训"，读xùn |
| 施 | 施读为移 | 通"移"，读yí |

**处理方法**：
- 查"读为字"的拼音
- 记录通假关系

### 2.4 模式4：类比注音

**格式**：`某如某`

**示例**：

| 原文 | 注音 | 现代拼音 |
|------|------|---------|
| 朝 | 朝音潮，如潮汐之潮 | cháo |

**处理方法**：
- 查"类比字"的拼音
- 结合现代词汇理解

## 三、提取流程

### 3.1 步骤1：搜索读音标注

**方法1：直接搜索地名+音注**

```bash
# 在《史记集解三家注索隐正义》中搜索
grep -E "音|反切" shiji_zhengyi.txt | grep "地名"
```

**方法2：搜索"正义"段落中的读音**

```bash
grep -B2 -A2 "正义.*音" shiji_zhengyi.txt
```

**方法3：搜索反切模式**

```bash
grep -E "[^,。]+[，。].*反[，。]" shiji_zhengyi.txt
```

### 3.2 步骤2：反切转现代拼音

**工具**：`scripts/fanqie_to_pinyin.py`（待实现）

**转换表**（部分示例）：

| 反切上字 | 声母 | 反切下字 | 韵母+调 | 结果 |
|---------|------|---------|---------|------|
| 于 | y | 达 | ān | yān |
| 古 | g | 侯 | ōu | gōu |
| 苦 | k | 浪 | àng | kàng |

**难点**：
- 中古音与现代音的对应关系
- 声调的转换规则
- 特殊韵母的处理

### 3.3 步骤3：验证在史记中出现

**目的**：确保提取的词条确实在《史记》原文中存在

**工具**：

```bash
# 验证词条是否在史记原文中出现
python scripts/verify_in_shiji.py "阏与"

# 统计出现次数
grep -c "阏与" corpus/shiji/史记*.txt
```

**判断标准**：
- 出现次数 ≥ 3：值得收录
- 出现次数 1-2：酌情收录
- 出现次数 0：删除（可能是注释用字）

### 3.4 步骤4：分类整理

**分类维度**：
1. **地名**：会稽、阏与、朝鲜等
2. **人名**：召公、伍员、华元等
3. **官职**：仆射、大行、庶长等
4. **族群**：冒顿、头曼、荤粥等
5. **其他**：特殊读音词汇

**数据格式**：

```json
{
  "text": "阏与",
  "pinyin": ["yān", "yǔ"],
  "context": "地名",
  "note": "[史记正义] 赵奢破秦军处，正义：'音焉与'",
  "source": "史记正义",
  "frequency": 5
}
```

## 七、常见问题

### Q1: 反切如何转换为现代拼音？

**A**: 基本规则：
1. 取上字声母
2. 取下字韵母+声调
3. 组合后查《汉语拼音方案》对应

**工具辅助**：
- 《广韵》查询工具
- 在线反切转换器
- 中古音韵母对照表

### Q2: 《史记正义》的注音都可靠吗？

**A**: 大部分可靠，但需要：
- 对照《史记索隐》《史记集解》
- 查证现代权威词典
- 结合《史记》原文语境

### Q3: 提取到的词条都要收录吗？

**A**: 需要筛选：
- ✅ 收录：pinyin-pro无法处理 + 出现≥3次
- ❌ 不收录：pinyin-pro已正确 或 仅出现1次

### Q4: 如何处理一字多音？

**A**:
- 优先使用固定词组（如"召公"，不用"召"单字）
- 记录context（地名/人名/官职）
- 结合原文语境判断
