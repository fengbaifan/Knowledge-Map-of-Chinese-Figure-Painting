## V0. 红线

### V0-A70. 严禁修改动词标注格式（2026-03-24，全书铁律，与 SKILL_03c1-v2-rules A70 同步）

任何已标注的动词格式**不得**再迁移/修改。特别是：

| 允许 | 禁止 |
|---|---|
| 补标未标注的动词 | 把 `⟦◈伐⟧` 改为 `⟦◉伐⟧` |
| 修复边界切分错误（如 `自〖[杀〗` 合并）| 把 `〖[X〗` 旧格式迁移到 `⟦◉X⟧` 新格式 |
| 新增动词条目 | 批量将 `〖[斩首〗` 改为 `⟦◉斩首⟧` |

**理由**：动词格式迁移容易引入隐蔽错误（如嵌套、分割不对），且全书已稳定。迁移应由专项工具有计划地推进，不得在反思任务中"顺手改"。

**扫描验证**：任何反思任务结束前必须 `grep -c '⟦[◈◉○◇]'` 前后计数一致（只增不减）。

## V.1 军事动词规律 ⟦◈⟧

### V1. 核心 17 个军事动词

`伐/攻/击/袭/侵/战/破/败/灭/围/下/拔/救/取/定/降/走`

### V2. 复合动词拆分（2026-03-24 起全书铁律）

不得写 `⟦◈击破⟧`、`⟦◈攻灭⟧` 这类复合动词，必须拆为独立单字：
```
❌ ⟦◈击破⟧〖◆秦〗
✅ ⟦◈击⟧⟦◈破⟧〖◆秦〗
```

**例外**：极少数语义黏合的固定词（如 `⟦◈下车⟧` 作为"就职"固定搭配）才可整体标注，需在规律库单独列出。

### V3. 消歧高频词

| 动词 | 消歧 | 语境 |
|---|---|---|
| 走 | `⟦◈走\|逃亡⟧` | 逃走/败走；与"行走"区分 |
| 取 | `⟦◈取\|攻取⟧` | 军事攻取；与"拿取"区分 |
| 定 | `⟦◈定\|平定⟧` | 平定某地；与"确定"区分 |
| 下 | `⟦◈下\|攻下⟧` | 攻下城邑；与"下行"区分 |

### V4. `〖[斩首〗` vs `⟦◉斩⟧`（2026-03-24）

**"斩首 N 级"** 是**军事战绩名词**（制度性表述），保留旧格式 `〖[斩首〗`。  
**单字"斩X"**（斩某人某官）是**刑罚动词**，用 `⟦◉斩⟧`。

```
〖[斩首〗十五级           ← 名词（战绩计数）
⟦◉斩⟧〖@颜良〗          ← 动词（处决某人）
```

## V.2 刑罚动词规律 ⟦◉⟧

### V5. 核心 15 个刑罚动词

`杀/诛/斩/弑/族/戮/刺 · 废/囚/执/笞/刑 · 反/亡/死`

### V6. "死" 的多义消歧

| 语境 | 标注 | 说明 |
|---|---|---|
| 自然死亡（薨/卒/病死）| 不标 | 未归为刑罚 |
| 处死/死刑 | `⟦◉死\|处死⟧` | 强调判处死罪 |
| 殉死/为...死 | `⟦◉死\|殉死⟧` | 主动赴死 |

### V7. 复合动词整体标注（2026-04-17，091 章）

某些刑罚含义的复合动作应整体标为 `⟦◉复合词⟧`，**不得拆分**：

| 正确 | 错误 | 典型出处 |
|---|---|---|
| `⟦◉自杀⟧` | `自〖[杀〗` / `自⟦◉杀⟧` | 全书 67 处；自裁含义的固定词 |
| `⟦◉阬杀⟧` | `⟦◉阬⟧⟦◉杀⟧` | 坑杀，整体语义 |
| `⟦◉追杀⟧` | `⟦◉追⟧⟦◉杀⟧` | 追赶并杀死的单一事件 |

**边界修复**（2026-04-17，091 章 L43）：
- 错切：`欲自〖[杀〗` → 正确：`欲⟦◉自杀⟧`
- 理由：`自` 是"自己"代词，与刑罚词"杀"合成"自杀"是整体动作

### V8. `〖[坐法〗` / `〖[论死〗` 等复合制度词（2026-04-16 确认）

`〖[X〗` 前缀的 2-3 字制度性动词保留为旧格式，不迁移（参 V0-A70）。例如：
`〖[坐法〗`、`〖[论死〗`、`〖[弃市〗`、`〖[诛灭〗`、`〖[斩首〗`、`〖[伏诛〗` 等。

## V.3 政治动词规律 ⟦○⟧

### V9. 政治动词核心清单（2026-04-17 扩展）

政治动词表示君主/权力机构对人/地/权位的安置与变动。

| 动词 | 含义 | 典型语境 |
|---|---|---|
| `⟦○立⟧` | 立某人为王/帝/侯 | `⟦○立⟧〖@X〗为〖;王〗` |
| `⟦○封⟧` | 分封土地/爵位 | `⟦○封⟧〖@X〗为〖;侯〗` |
| `⟦○拜⟧` | 拜任官职 | `⟦○拜⟧〖@X〗为〖;丞相〗` |
| `⟦○赐⟧` | 赏赐（爵/金/邑）| `⟦○赐⟧〖@X〗爵〖#列侯〗` |
| `⟦○益⟧` | 增益食邑/封户 | `⟦○益⟧〖○食⟧〖$X户〗` |
| `⟦○食⟧` | 享有食邑 | `⟦○食⟧〖=邑名〗〖$X户〗` |
| `⟦○谥⟧` | 赐谥号 | `⟦○谥⟧为〖@X侯〗` |
| `⟦○废⟧` | 废黜（君主/太子）| `⟦○废⟧〖#太子〗` |
| `⟦○嗣⟧` | 嗣立（继位）| `⟦○嗣⟧〖;侯〗` |
| `⟦○朝⟧` | 朝见/入朝 | `⟦○朝⟧〖◆魏〗` |
| `⟦○崩⟧` | 天子/皇帝死亡 | `〖#帝〗⟦○崩⟧` |
| `⟦○薨⟧` | 诸侯/大臣死亡 | `〖;丞相〗⟦○薨⟧` |
| `⟦○卒⟧` | 士大夫死亡 | `〖@X〗⟦○卒⟧` |
| `⟦○举⟧` | 推举贤才（选拔）| `⟦○举⟧〖@贤〗` |
| `⟦○荐⟧` | 推荐某人 | `⟦○荐⟧〖@X〗` |
| `⟦○辟⟧` | 辟除（征召入幕）| `⟦○辟⟧〖@X〗为〖;掾〗` |

**君主死亡三词（崩/薨/卒）说明**（2026-04-17 新增）：
- **为什么是政治动词**：君主/诸侯/大臣的死亡是**权力更替事件**，触发嗣位、国除、谥号等一系列政治安排，是 KG 中关键的时间锚点
- **与刑罚动词"死"的区分**：
  - `⟦◉死⟧` / `⟦◉死|处死⟧` 表处决或主动赴死（刑罚结果）
  - `⟦○崩⟧` `⟦○薨⟧` `⟦○卒⟧` 表自然死亡/事件节点（政治事件）
- **与家族动词的关系**：君主死亡**同时是谱系事件**，可选双标 `⟦○崩⟧⟦♂崩⟧`，但默认使用单一 `⟦○⟧` 避免冗余
- **扫描模式**：`(?<!⟦○)崩/薨/卒(?![a-zA-Z])` — 全书候选约 2000+ 处

**选拔/推举并入政治说明**（2026-04-17）：
举/荐/辟 原考虑独立分类，最终并入 `⟦○⟧` 政治，因三者都是**君主/权臣对人才的安置行为**，语义与拜/封同类。

### V10a. `所食` 的语境分化（2026-04-17 订正，030/043/117 章）

"所食" 不能盲目标 `所⟦○食⟧`，必须按语境分化：

| 语境 | 正确 | 理由 |
|---|---|---|
| 除前**所食** (邑/地名) | `除前所⟦○食⟧` | ✓ 封邑的所食（享有食邑）|
| **无所食** / 无可食 | `无所食`（不标）| ✗ "吃"/"食物"义，非政治 |
| 尝**所食** 饿人 | `尝所食桑下`（不标）| ✗ "供食、给吃"义 |
| 民**无所食** | `民无所食`（不标）| ✗ 民众无粮可吃 |

**安全正则**：`所⟦○食⟧` 只应出现在 `除前所⟦○食⟧` 固定搭配中，**前面必须有"前"字**。

```python
# 错误：
(r'(?<!⟦)所食', r'所⟦○食⟧')
# 正确：
(r'前所食', r'前所⟦○食⟧')
```

### V10. `⟦○食⟧` 的三种用法（2026-04-17，095 章）

"食" 作政治动词有三个固定复合：

| 模式 | 标注 | 例 |
|---|---|---|
| 益/定/更/所食 | `益⟦○食⟧` `定⟦○食⟧` `更⟦○食⟧` 所⟦○食⟧` | `益⟦○食⟧〖$千户〗` |
| 食 + 地名 | `⟦○食⟧〖=地名〗` | `⟦○食⟧〖=汝阴〗` |
| 食邑 + 地名/户数 | `⟦○食邑⟧〖=地名〗` | `⟦○食邑⟧〖=涿〗〖$五千户〗` |

**排除语境**：
- 饮食（如 `饮食`、`食客`、`食肉`）→ 不标
- 粮食（如 `兵食`、`食尽`）→ 不标

### V11. `⟦○立⟧` 的政治语境（2026-04-17，85 章共 483 处补标）

仅在**立人为王/帝/侯**或**拥立皇帝**语境下标 `⟦○立⟧`：

```
✅ 共⟦○立⟧〖;代王〗为〖@孝文皇帝〗
✅ ⟦○立⟧〖@沛公〗为〖;汉王〗
❌ 他立在门口 （日常动作"立"，不标）
❌ 嗣立 （组合词，见 V9 "嗣" 单独处理）
```

**安全扫描模式**：
- `(?<!⟦○)(?<![已即])立〖[@;#]` → `⟦○立⟧`
- `(?<!⟦○)(?<![已即嗣])立为〖[;@#]` → `⟦○立⟧为`

### V12. `⟦○拜⟧` 的政治语境（2026-04-17，53 章共 131 处补标）

在**拜为官职**或**拜某人为官**语境下标 `⟦○拜⟧`：

```
✅ ⟦○拜⟧〖@婴|灌婴〗为〖;郎中〗
✅ ⟦○拜⟧为〖;博士〗
❌ 跪拜 / 朝拜 / 礼拜 （日常动作，不标）
```

### V13a. "X + 兵/卒" 军事复合的避标（2026-04-17，005 章订正）

**禁止**把 `发兵/起兵/引兵/罢兵/聚兵/收兵/徵兵` 这类**军事复合动作**标为政治动词，哪怕前字"罢"单用是政治。

| 错误 | 正确 |
|---|---|
| `⟦○罢⟧兵归` | `罢兵归` 或 `〖[罢兵〗归`（旧格式）|
| `⟦○发⟧兵` | `发兵` 或 `〖[发兵〗`（旧格式）|

**原因**：这些 "X兵/X卒" 是**军事事件名词**，全书已有 103 处用旧格式 `〖[X兵〗` 且大量未标的惯例。单字"罢"在政治语境（罢官/罢相/罢郡）才作 `⟦○罢⟧`。

**安全正则**：`(?<!⟦○)罢(?=〖[;=#^])` 而**不要**加 `罢兵/罢卒` 固定搭配。

### V13. 并列政治动词的单独标注（2026-04-17，095 章）

"赐益"、"赐食"、"废立" 等双动词并列，应**各自单独**标注：

```
✅ ⟦○赐⟧⟦○益⟧〖@X〗邑
❌ ⟦○赐益⟧〖@X〗邑  （不应合并）

✅ 〖:赐食〗邑 → ⟦○赐食⟧邑 （此处 `赐食` 作为固定复合动作可整体标，但"赐食+邑"仍需分解）
```

### V14. `⟦○国除⟧` 整词标注（2026-04-17，25 章共 430 处统一）

"国除"是**封国废除**的固定政治事件，作为**复合政治动词整体标**：

```
✅ 〖@X〗有罪，⟦○国除⟧。
✅ 无后，⟦○国除⟧。
❌ 〖#国〗除  （不拆为"国+除"）
❌ ⟦○国⟧⟦○除⟧  （不拆为两个动词）
```

**理由**：
- "国"指代特定封国（这个侯国），不是泛指
- "除"是动作（废除）
- 两字共同构成固定政治事件表述
- 年表章（017-021）高频出现，占 396 处

| `⟦⇨奔⟧` | 奔逃（非军事）| `奔〖◆晋〗`（往国外）|
| `⟦⇨亡⟧` (wáng) | 逃亡（非刑罚）| `亡〖=地〗`（与刑罚"亡"区分）|
| `⟦⇨归⟧` | 归葬/归故里 | 注意与政治"归附"区分 |

### V26. 与军事动词的区分

| 词 | 军事 ⟦◈⟧ | 位移 ⟦⇨⟧ |
|---|---|---|
| 走 | 溃败之走：`⟦◈走\|逃亡⟧` | — |
| 奔 | — | 政治奔国：`⟦⇨奔⟧〖◆晋〗` |
| 徙 | — | 迁徙：`⟦⇨徙⟧〖#民〗` |
| 亡 | 战场逃亡：`⟦◈亡⟧` | 政治流亡：`⟦⇨亡⟧〖=地〗` |

### V27. `⟦○迁⟧` vs `⟦⇨迁⟧` 的细分

- **政治迁** `⟦○迁⟧`：迁官（升/贬某人职位），如 `迁〖;丞相〗`
- **位移迁** `⟦⇨迁⟧`：地理迁移，如 `迁〖=都〗`、`迁〖#民〗〖=地〗`

**扫描正则**：
```
迁〖[;#][^◆=]  → ⟦○迁⟧（官职/人群，政治）
迁〖[=◆]     → ⟦⇨迁⟧（地名/邦国，位移）
```
