# SKILL 02f: 文本统计 — 定量分析与分布特征

> 对标注文本进行多维度定量统计，揭示《史记》的文本规模、结构特征和实体分布规律。
> 本文专注**基础文本层**指标；深度**语义层**统计（字级标注覆盖率、实体条目、专题覆盖）参见 [SKILL_08b1 标注完成情况统计](./references/SKILL_08b1_标注完成情况统计.md)。

---

## 一、统计维度

### 1.1 基础规模（2026-04-18 数据）

| 指标     | 数值             | 数据源 |
| -------- | ---------------- | ------ |
| 总章节数 | 130 篇 | `chapter_md/*.tagged.md` |
| 纯汉字   | 576,900 字 | `compute_annotation_coverage.py` |
| 含标点   | ~706,000 字 | 手工估算 |
| 实体词条 | 15,674 个（18+4 类）| `build_entity_index.py` |
| 标注次数 | 130,177 次 | `build_entity_index.py` |
| 字级标注率 | 38.01%（已标 219,259 / 总 576,900）| `compute_annotation_coverage.py` |

### 1.2 五体分布

| 体裁   | 篇数   | 字数    | 占比   |
| ------ | ------ | ------- | ------ |
| 本纪   | 12     | 76,730  | 13.3%  |
| 表     | 10     | 70,525  | 12.2%  |
| 书     | 8      | 41,761  | 7.2%   |
| 世家   | 30     | 132,853 | 23.0%  |
| 列传   | 70     | 255,046 | 44.2%  |

### 1.3 可统计指标

| 类别     | 指标                     | 工具                        |
| -------- | ------------------------ | --------------------------- |
| 文本长度 | 章节字数、段落字数、句长 | `compute_annotation_coverage.py`（按章节字数分布）|
| 词频     | 字频、实体频次、虚词频次 | `analyze_word_frequency.py` |
| 实体密度 | 每百字实体数、分类占比   | `compute_annotation_coverage.py` + `build_entity_index.py` |
| 对话占比 | 引文字数/总字数          | 待补脚本 |
| 标点密度 | 每百字标点数             | 待补脚本 |

## 二、工具

```bash
# 基础：字数与字级标注覆盖率
python scripts/compute_annotation_coverage.py

# 词频
python scripts/analyze_word_frequency.py

# 实体条目与出现次数
python kg/entities/scripts/build_entity_index.py
```

> ⚠️ 历史脚本 `scripts/analyze_tagged.py` 已于 2026-04-18 DEPRECATED（路径硬编码失效、正则漏 `◆邦国` 与动词类），见文件头注释。
