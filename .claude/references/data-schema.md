# 数据结构详细规范

本文档提供中国人物画知识图谱数据结构的详细说明。

**主规范文档**：[01-元知识/中国人物画知识图谱数据结构v1_4.md](../../01-元知识/中国人物画知识图谱数据结构v1_4.md)

## 溯源字段详细规范

所有表必须包含以下溯源字段：

| 字段 | 必填性 | 格式规范 | 示例 |
|------|--------|----------|------|
| `source_book` | 必填 | 使用书名号《》 | 《历代名画记》 |
| `source_chapter` | 强烈推荐 | 格式：卷次·章节名 | "卷一·顾野王" |
| `source_text` | 推荐 | 保留原文标点，使用""标注引文 | "顾野王，吴郡人，善画" |
| `source_page` | 可选 | 页码或版本信息 | "第12页"、"四库本卷三" |

## ID 命名规范详细说明

### 时序本体 (M1.1)
- **前缀**：`period_`
- **格式**：`period_[朝代/时期]`
- **示例**：
  - `period_tang`：唐代
  - `period_song_north`：北宋
  - `period_five_dynasties`：五代

### 空间地理本体 (M1.2)
- **前缀**：`loc_`
- **格式**：`loc_[地名拼音]`
- **示例**：
  - `loc_changan`：长安
  - `loc_luoyang`：洛阳
  - `loc_jiankang`：建康

### 图像志题材分类 (M1.3)
- **前缀**：`icon_`
- **格式**：`icon_[题材类型]`
- **示例**：
  - `icon_portrait`：人物画
  - `icon_landscape`：山水画
  - `icon_flower_bird`：花鸟画

### 历史人物 (M2.1)
- **前缀**：`meso_`
- **格式**：`meso_[姓名拼音]`
- **示例**：
  - `meso_gu_kaizhi`：顾恺之
  - `meso_wu_daozi`：吴道子
  - `meso_zhang_sengyou`：张僧繇

### 履历与时空轨迹 (M2.2)
- **前缀**：`cv_`
- **格式**：`cv_[序号]`
- **示例**：
  - `cv_001`
  - `cv_002`

### 社会关系实例 (M2.3)
- **前缀**：`rel_`
- **格式**：`rel_[序号]`
- **示例**：
  - `rel_001`
  - `rel_002`

### 作品实体 (M3.1)
- **前缀**：`micro_`
- **格式**：`micro_[作品名拼音]`
- **示例**：
  - `micro_luoshen_fu`：洛神赋图
  - `micro_nushi_zhen`：女史箴图
  - `micro_tianwang_song_zi`：天王送子图

### 文献著录 (M3.2)
- **前缀**：`lit_`
- **格式**：`lit_[序号]`
- **示例**：
  - `lit_001`
  - `lit_002`

## 字段类型说明

### 文本字段
- **短文本**：人名、地名、作品名等（<100字符）
- **中文本**：章节名、简短描述（100-500字符）
- **长文本**：原文引用、详细描述（>500字符）

### 数值字段
- **年份**：YYYY 格式，如 618、907
- **年份范围**：YYYY-YYYY 格式，如 618-907
- **序号**：整数，从 1 开始

### 布尔字段
- **格式**：TRUE/FALSE 或 1/0
- **用途**：标记字段，如 is_verified

### 日期字段
- **格式**：YYYY-MM-DD
- **用途**：精确日期记录

## 字段命名规范

### 通用规则
- 使用小写字母和下划线
- 避免使用缩写（除非是通用缩写）
- 使用描述性名称

### 常见字段后缀
- `_id`：主键标识符
- `_ref`：外键引用
- `_name`：名称字段
- `_text`：文本内容
- `_date`：日期字段
- `_year`：年份字段
- `_type`：类型字段

### 溯源字段前缀
- `source_`：所有溯源相关字段统一使用此前缀

## 表间关系说明

### 一对多关系
- 一个人物 → 多条履历记录
- 一个人物 → 多个作品
- 一部文献 → 多条著录记录

### 多对多关系
- 人物 ↔ 人物（社会关系）
- 作品 ↔ 文献（著录关系）

### 层级关系
- M1（宏观）→ M2（中观）→ M3（微观）
- 时序本体 → 人物 → 作品
- 空间本体 → 人物 → 履历

## 数据完整性约束

### 必填字段约束
- 所有表必须有 `source_book` 字段
- 所有实体表必须有主键 ID
- 所有关系表必须有外键引用

### 唯一性约束
- 主键 ID 在表内必须唯一
- 人物名称 + 朝代组合应尽量唯一

### 引用完整性约束
- 外键必须指向已存在的主键
- 删除主记录前必须先删除引用记录

## 数据质量标准

### 完整性标准
- 必填字段覆盖率：100%
- 推荐字段覆盖率：>80%
- 可选字段覆盖率：>50%

### 准确性标准
- 书名格式正确率：100%
- ID 前缀正确率：100%
- 枚举值正确率：100%

### 一致性标准
- 同一人物在不同表中的 ID 一致
- 同一作品在不同表中的名称一致
- 同一文献的书名格式一致
