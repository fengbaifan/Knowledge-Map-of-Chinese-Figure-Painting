# 格式规范

本文档提供各字段的格式要求与正确/错误示例。

---

## 溯源字段格式

### source_book（来源典籍）

**格式要求**：使用中文书名号《》

**正确示例**：
```
《历代名画记》
《宣和画谱》
《古画品录》
《续画品》
《贞观公私画史》
```

**错误示例**：
```
❌ 历代名画记          # 缺少书名号
❌ <历代名画记>        # 错误符号
❌ "《历代名画记》"    # 多余引号
❌ 《历代名画记        # 缺少右书名号
```

---

### source_chapter（来源章节）

**格式要求**：卷次·章节名（使用中文间隔号·）

**标准格式**：
```
卷一·顾野王
卷三·西京寺观等画壁·兴善寺
卷五·论画六法
序·论画体工用拓写
```

**可接受格式**：
```
⚠️ 卷一 顾野王        # 空格分隔，可自动修复为·
⚠️ 卷五              # 无具体章节名，可接受
```

**错误示例**：
```
❌ 第一卷·顾野王      # 应为"卷一"而非"第一卷"
❌ 卷1·顾野王         # 应为"卷一"而非"卷1"
```

---

### source_text（原文引用）

**格式要求**：保留原文标点，使用""标注引文

**正确示例**：
```
"赵孟頫，字子昂，号松雪道人，吴兴人。宋宗室，入元官至翰林学士承旨，荣禄大夫。善书画，精鉴赏，书画之学，冠绝当代。"

"顾恺之《洛神赋图》，曹植洛神赋，恺之为之图，人物秀丽，山水幽深。"
```

**省略格式**：
```
"赵孟頫，字子昂...善书画，精鉴赏..."
```

**长度建议**：50-500字，核心信息完整即可

---

### source_page（页码版本）

**格式要求**：自由格式，建议包含版本信息

**示例**：
```
第12页
四库本卷三
中华书局1985年版第45页
页23-24
```

## ID字段格式

### person_id（人物ID）

**格式要求**：`meso_` + 拼音或英文

**正确示例**：
```
meso_gu_kaizhi
meso_wu_daozi
meso_zhao_mengfu
meso_zhang_sengyao
```

**错误示例**：
```
❌ gu_kaizhi          # 缺少meso_前缀
❌ meso_顾恺之        # 不应使用中文
❌ MESO_gu_kaizhi    # 前缀应小写
```

---

### work_id（作品ID）

**格式要求**：`micro_` + 拼音或英文

**正确示例**：
```
micro_luoshen_fu
micro_nushi_zhen
micro_wangchuan_tu
```

---

### period_id（时序ID）

**格式要求**：`period_` + 朝代或时期

**正确示例**：
```
period_tang
period_song
period_yuan
period_ming
period_northern_song
```

---

### location_id（地点ID）

**格式要求**：`loc_` + 地名拼音

**正确示例**：
```
loc_changan
loc_luoyang
loc_jiankang
loc_bianjing
```

---

### icon_id（图像志ID）

**格式要求**：`icon_` + 题材类型

**正确示例**：
```
icon_portrait
icon_landscape
icon_bird_and_flower
icon_religious
```

---

### cv_id（履历ID）

**格式要求**：`cv_` + 数字编号

**正确示例**：
```
cv_001
cv_002
cv_123
```

---

### rel_id（关系ID）

**格式要求**：`rel_` + 数字编号

**正确示例**：
```
rel_001
rel_002
rel_456
```

---

### lit_id（文献ID）

**格式要求**：`lit_` + 数字编号

**正确示例**：
```
lit_001
lit_002
lit_789
```

## 枚举字段格式

### primary_role（人物角色）

**允许值**：
```
Painter          # 画家
Critic           # 评论家
Collector        # 收藏家
Calligrapher     # 书法家
Scholar          # 学者
Patron           # 赞助人
Other            # 其他
```

**错误示例**：
```
❌ 画家            # 应使用英文
❌ painter        # 首字母应大写
❌ PAINTER        # 不应全大写
```

---

### status（作品状态）

**允许值**：
```
Extant           # 存世
Lost             # 佚失
Attributed       # 传
Copy             # 摹本
Unknown          # 未知
```

**常见错误**：
```
❌ 存世            # 应使用英文Extant
❌ 佚失            # 应使用英文Lost
❌ 存世(传摹)      # 应拆分为status和其他字段
```

---

### support（作品材质）

**允许值**：
```
Silk             # 绢本
Paper            # 纸本
Wall             # 壁画
Wood             # 木板
Other            # 其他
Unknown          # 未知
```

---

### relation_type（关系类型）

**允许值**：
```
Teacher          # 师
Student          # 徒
Friend           # 友
Colleague        # 同僚
Spouse           # 配偶
Parent           # 父母
Child            # 子女
Sibling          # 兄弟姐妹
Other            # 其他
```

## JSON字段格式

### birth_death（生卒年）

**格式要求**：JSON对象，包含birth和death字段

**正确示例**：
```json
{"birth": 535, "death": 602, "period_ref": "period_chen"}
{"birth": 508, "death": 555}
{}  # 生卒年不详时使用空对象
```

**错误示例**：
```
❌ {birth: 535, death: 602}           # 键名应加引号
❌ {"birth": "535", "death": "602"}   # 年份应为数字而非字符串
❌ 535-602                            # 应为JSON格式
```

---

### time_range（时间范围）

**格式要求**：JSON对象，包含start和end字段

**正确示例**：
```json
{"start": 618, "end": 907}
{"start": -206, "end": 220}  # 负数表示公元前
```

---

### authority_ids（权威标识符）

**格式要求**：JSON对象，键为数据库名，值为ID

**正确示例**：
```json
{"Wikidata": "Q7269", "CBDB": "32145"}
{"Wikidata": "Q197259"}
{}  # 无权威标识符时使用空对象
```

---

## 特殊字段格式

### other_names（其他名称）

**格式要求**：JSON数组

**正确示例**：
```json
["北绛郡公"]
["小字七符", "湘东郡王", "梁元帝"]
[]  # 无其他名称时使用空数组
```

---

### visual_elements（视觉元素）

**格式要求**：JSON数组，描述图像志的视觉特征

**正确示例**：
```json
["人物", "衣冠", "面部特征"]
["山水", "树木", "建筑"]
```

---

### dynastic_info（朝代信息）

**格式要求**：JSON对象，包含朝代详细信息

**正确示例**：
```json
{"dynasty": "唐", "emperor": "太宗", "era_name": "贞观"}
{"dynasty": "宋", "period": "北宋"}
```

---

## 文本字段格式

### name（姓名）

**格式要求**：中文全名，2-4字

**正确示例**：
```
顾恺之
吴道子
赵孟頫
张僧繇
```

---

### title（作品标题）

**格式要求**：中文标题，不加书名号

**正确示例**：
```
洛神赋图
女史箴图
王川图
步辇图
```

**错误示例**：
```
❌ 《洛神赋图》    # 作品标题不加书名号
```

---

### quote（引文）

**格式要求**：保留原文标点和格式

**正确示例**：
```
"画有六法，真仙为难。"
"笔力遒劲，风骨峻峭。"
```
